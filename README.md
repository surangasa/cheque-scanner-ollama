# cheque-scanner-ollama
Advanced PyQt6 desktop app for batch extraction of data from scanned GCC/Qatari bank cheques using Ollama vision LLMs (Gemma4, Llama 3.2 Vision etc.). Features: ROI cropping for amount/MICR/payer/date, robust verification, Arabic handwriting escalation, handover payment detection, checkpoint/resume, Tesseract fallback, PDF batch processing.
